# subsound

A subsonic music player.

## Goals

- [ ] A nice, smooth interface
- [ ] Browsing content
- [ ] Offline support
- [ ] Audio playback
- [ ] Chromecast
- [ ] Airplay
- [ ] Linux

Possible goals:
 - Chromecast
 - Airplay
 - Support other servers than Subsonic compatible APIs
 
Non-goals:
 - Video support
